# sum-of-all-primes
A sum of all primes program written by Clojure.
Find the sum of all prime numbers below two million.

## Usage

1. Build
```
lein uberjar
```
2. Run
```
java -jar target/sum-of-all-primes-0.1.0-SNAPSHOT-standalone.jar
```
3. Unit test

```
lein test sum-of-all-primes.core-test
```
